# Lavaz Studio

A dynamic web app builder powered by Lavaz Life Designs™ visuals. Upload assets, define app specs, generate & download complete responsive projects.

## Quick Start
1. Open `index.html` in any browser.
2. Fill the builder form with your app idea.
3. Preview the generated app.
4. Download ZIP bundle.

## Features
- Responsive design (mobile-first).
- Asset integration from uploads.
- Dynamic generation previews.
- Themes inspired by LLD/Lavaz cyber-metal aesthetic.
- Extensible for AI integration (add OpenAI API in app.js).

## Customization
- Replace `/assets/` with your files.
- Extend `app.js` generation logic for complex apps.
- Add JSZip CDN for true ZIP exports: `<script src="https://cdnjs.cloudflare.com/ajax/libs/jszip/3.10.1/jszip.min.js"></script>`

## Tech Stack
- Vanilla HTML/CSS/JS (no frameworks for lightweight).
- Orbitron font for futuristic vibe.
- Fully responsive with CSS Grid/Flexbox.
