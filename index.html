🚀 Deploy Your App

Deploy via GitHub API. Requires a Personal Access Token with repo scope from github.com/settings/tokens.

Deploy via Netlify API. Get a Personal Access Token from app.netlify.com/user/applications.

⚡ Netlify Drop

1. Click "Download" below to get your HTML file
2. Open app.netlify.com/drop
3. Drag & drop your file — live in seconds, no account needed

▲ Vercel Deploy

1. Download your app
2. Visit vercel.com/new
3. Import from GitHub or drag your project folder

📖 GitHub Pages (Manual)

1. Create a new GitHub repository
2. Upload your index.html to the root
3. Settings → Pages → Deploy from branch → main
